# 🤖 J.A.R.V.I.S – Just A Rather Very Intelligent System

> 🚀 A Powerful AI Voice Assistant built using Python that listens, understands, and executes commands like a real virtual assistant.

---

## 🌟 Introduction

J.A.R.V.I.S (Just A Rather Very Intelligent System) is an advanced voice-controlled AI assistant inspired by the iconic system from Iron Man.  
This project demonstrates how Artificial Intelligence, Speech Recognition, and Automation can be combined to build a smart assistant capable of performing real-world tasks.

With a simple voice command, JARVIS can control your system, access the internet, fetch information, and even communicate with users — making it a powerful productivity tool.

## ✨ Features

### 🎙️ Voice Intelligence
- Converts speech into text using Google Speech Recognition
- Supports natural human commands
- Continuous listening mode

### 🔊 Smart Voice Response
- Uses pyttsx3 for offline Text-to-Speech
- Customizable voice (Male/Female)

### 🌐 Web Automation
- Open popular websites:
  - YouTube  
  - Google  
  - Instagram  
  - Facebook  
  - WhatsApp Web  

### 📚 Knowledge Engine
- Fetches summarized data from Wikipedia
- Provides quick and accurate information

### 💻 System Control
- Open/Close applications:
  - Notepad  
  - Command Prompt  
  - Chrome  
- Perform OS-level operations

### 🎵 Entertainment
- Plays random music from local directory
- YouTube music playback using voice

### 📷 Camera Integration
- Opens webcam using OpenCV
- Live video streaming with exit control

### 📩 Communication
- Send WhatsApp messages automatically
- Voice-triggered messaging system

### 🌍 Utilities
- Get system IP address
- Smart greetings based on time
- Basic assistant responses
